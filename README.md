# 🌐 Boxing House PF — Sistema de academia

![Visualização do Site](./boxing.gif)

## 🧠 Sobre o Projeto

Plataforma completa para gestão da academia Boxing House online disponível em: [https://boxinghouse.com.br/](https://boxinghouse.com.br/)

----

## ⚙️ Tecnologias Utilizadas

- LARAVEL (back-end)  
- SQLITE como banco de dados  
- TailwindCSS para estilização  
- Hospedado em servidor Linux (Ubuntu + Nginx)  
- Versionamento com Git + GitHub
